## Инструкция По Запуску:


+ ``` Clone the repo```


+ ``` cd laravel-car-loan-api-test-task```


+ ```cp .env.example .env```


+ ```docker-compose build```


+ ```docker-compose up -d```


+ ```docker-compose run -it composer install``` 


+ ```docker-compose run -it app php artisan migrate --seed```
